앱 이름:
TagWeaver MP3/FLAC 태그 편집기

간단한 설명:
MP3/FLAC 태그·평점·앨범 아트·가사를 오프라인으로 편집해요.

랜딩 부제:
MP3/FLAC 오프라인 태그 편집기

자세한 설명:
MP3/FLAC 오프라인 태그 편집기예요

TagWeaver는 MP3와 FLAC 파일의 태그를 직접 수정하는 오프라인 편집기예요.
제목, 아티스트, 앨범, 앨범 아티스트, 트랙 번호, 디스크 번호, 연도, 장르, 작곡가, 평점을 수정할 수 있어요.

주요 기능

* MP3 ID3 태그 편집
* FLAC Vorbis Comment 태그 편집
* 트랙 번호, 디스크 번호, 연도 편집
* 작곡가, 장르, 평점 등 메타데이터 수정
* 앨범 아트 추가 및 제거
* 가사 입력 및 수정
* 여러 파일 선택 및 일괄 태그 편집

여러 파일을 한 번에 수정해요

같은 앨범의 곡을 정리하거나 여러 MP3/FLAC 파일의 정보를 한 번에 변경할 수 있어요.
여러 파일 일괄 저장 기능은 Pro에서 사용할 수 있어요.

안전한 오프라인 태그 수정

TagWeaver는 완전한 오프라인 MP3/FLAC 태그 편집 도구예요.

* 로그인 없음
* 계정 생성 없음
* 파일 업로드 없음
* 광고 및 추적 없음

모든 태그 수정은 사용자 기기 내에서만 처리해요.

TagWeaver는 자동 정리 대신, 직접 수정과 예측 가능한 저장에 집중한 MP3/FLAC 오프라인 태그 편집기예요.

랜딩 페이지:
MP3/FLAC 태그, 평점, 앨범 아트, 가사를 직접 확인하고 정리하는 편집기입니다.

TagWeaver는 제목, 아티스트, 앨범, 트랙 번호, 연도, 장르, 평점처럼 음악 파일 안에 저장된 정보를 사용자가 원하는 방식으로 수정할 수 있게 만든 도구입니다.

앨범 아트와 가사도 같은 화면에서 정리할 수 있습니다. 자동으로 추측해 바꾸는 방식이 아니라, 사용자가 확인한 내용만 파일에 저장합니다.

여러 파일을 함께 선택해 같은 앨범의 곡 정보를 맞추거나, 흩어진 MP3/FLAC 파일의 기본 정보를 한 번에 정리할 수 있습니다. 여러 파일 일괄 저장은 Pro에서 사용할 수 있습니다.

지원하는 작업

• 태그 필드 수정
• 앨범 아트 정리
• 가사 입력과 수정
• 평점 메타데이터 편집
• MP3 ID3 태그 지원
• FLAC Vorbis Comment 지원
• 여러 파일 선택
• 일괄 편집

TagWeaver는 로그인, 계정 생성, 파일 업로드 없이 동작합니다. 모든 편집은 기기 안에서 처리되며, 파일은 사용자가 선택한 위치에 그대로 남습니다.

음악 라이브러리를 자동으로 관리하는 앱이 아니라, 이미 가지고 있는 MP3/FLAC 파일의 정보를 정확히 고치는 데 집중한 편집기입니다.
